# Node.js


## Prerequisite

- MongoDB
- Node

## Installing 
```shell
$ git clone https://github.com/hyewondev/node_api.git

$ cd node_api

$ npm install

$ cp .env.example .env
```
> Copy and paste your mongodb uri into .env


## Run
```shell
$ npm run migrate

$ npm start
```

## Test
```shell
$ npm test
```
